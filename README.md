# Enterprise Document Forensics Lab 

A multi-task deep learning framework and Streamlit application designed to detect visual anomalies and document tampering using 6-Channel fused inputs (RGB + Error Level Analysis).

##  How to Run the Application

### 1. Clone or Download the Project
Extract the zip file or clone this repository to your local directory.

### 2. Set Up a Virtual Environment (Recommended)
